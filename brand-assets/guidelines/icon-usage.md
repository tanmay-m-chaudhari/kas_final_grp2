# Icon Usage Guidelines

All icons in the ShopPilot design system use a 24×24 grid with a 2px stroke weight.

## Approved Icon Sets

- Lucide Icons (primary)
- Custom ShopPilot icons (shopping bag variants, trust badges)

## Rules

1. Never mix icon sets within the same component
2. Icons must always have an accessible aria-label or title
3. Fill icons are reserved for active/selected states only
4. Icon colour must meet 4.5:1 contrast ratio against its background
